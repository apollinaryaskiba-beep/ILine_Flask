# Проект от компании «ILine»: система управления персоналом
Веб-приложение на основе Flask и SQLAlchemy, отображающее информацию о сотрудниках из базы данных и предоставляющее функционал поиска и обновления данных.

# Функционал
- **Отображение**: ФИО, должности, даты приема, зарплаты и руководителя.
- **Сортировка**: двусторонняя сортировка (ASC/DESC) по всем полям.
- **Поиск**: по ФИО (полное или частичное совпадение).
- **Управление**: возможность смены руководителя через интерфейс с проверкой прав.
- **Безопасность**: защита от циклических зависимостей (подчиненный не может стать боссом своего начальника).
- **Иерархия**: обязательность руководителя для всех, кроме CEO.

# Оптимизация
Для стабильной работы с 50 000 записей использован лимит выборки (Limit 500) для предотвращения зависания браузера.

# Установка и запуск
1) Клонируем репозиторий:
```
    git clone <ссылка_на_ваш_репозиторий>
    cd Iline_Flask
```
2) Устанавливаем виртуальное окружение:
```
    python3 -m venv env
    source env/bin/activate
```
3) Устанавливаем зависимости:
```
    pip install -r requirements.txt
```
4) Настраиваем базу данных:
```
    DB_USER = "postgres"
    B_PASSWORD = "3131"
    B_NAME = "postgres"
```
5) Запускаем приложение:
```
    python app.py
    Приложение будет доступно по адресу: http://127.0.0.1:5000
```
